# About
Scrapy Demo Project  
for A Data Pro summer Internship Burgas 2022
# Install
requirements:
- python >= 3.9
- [poetry](https://python-poetry.org)
```bash
git clone git@github.com:simeonmartev/crawl_licitatie.git
cd crawl_licitatie
poetry install
```

# Crawl

## list spiders
```bash
cd crawl_licitatie
scrapy list
```

## crawl
```bash
poetry shell
cd crawl_licitatie
scrapy crawl api_spider
```
